# page-leisu-guide

本仓库用于归档和发布多个独立 HTML 页面。这些页面主要作为功能演示、工具说明或操作指引的静态文件，不绑定任何特定域名或网站。

## 项目简介

page-leisu-guide 是一个以 HTML 页面为主的归档仓库。所有页面均为独立文件，可直接在浏览器中打开，无需后端服务支持。仓库内容可能包含但不限于：

- 功能演示页面
- 操作指引页面
- 工具说明页面
- 其他静态展示页面

## 目录说明

```
page-leisu-guide/
├── README.md          # 本文件
├── pages/             # 存放所有 HTML 页面
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
└── assets/            # 页面所需的静态资源（如样式、脚本、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：所有独立 HTML 页面均放置于此目录下，文件名应反映页面用途。
- `assets/`：存放页面引用的外部资源文件，按类型分目录管理。

## 页面归档说明

- 每个 HTML 页面均为独立完整文件，可直接通过浏览器打开。
- 页面之间不存在依赖关系，可单独使用。
- 页面内容可能随着时间更新或替换，历史版本可通过 Git 提交记录查看。
- 不建议本仓库内容被用于商业或营销目的。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/page-leisu-guide.git
   ```
2. 进入 `pages/` 目录，双击任意 HTML 文件即可在浏览器中查看。
3. 也可通过 GitHub Pages 或其他静态托管服务部署整个仓库，以便在线访问。
